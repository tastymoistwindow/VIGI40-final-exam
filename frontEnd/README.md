
#Chess club player administration portal 

Chess club member administration portal is a platform for managing chess club member information.


## Features

- View a list of all registered players: accessible by clicking on "Player List" tab;
- Add a new player: accessible by clicking on "New player registration" tab.
- Edit information of an existing player: accessible by clicking "Edit" button found in the same row as the subject player's information. 
- Delete players that are no longer part of the club: accessible by clicking "Delete" button found in the same row as the subject player's information. 

- New player registration and existing player information amendment forms feature 3 buttons:
    - Submit: submits data entered into the form;
    - Clear Form: Clears data entered into the form;
    - Cancel: Aborts the form submission, returns user to "Player list" tab
